# M133_TypeScript-project

## Installing
To install all dependencies for the project, this code must be executed in the same directory as the project:
```
npm install
```
You must also create a folder called build.

## Used VS Code Extentions
* Mocha Test Explorer
* Test Explorer UI
* ESLint
